1. Persiapan & Inisialisasi

Untuk memulai menggunakan Library UI TripleSixxx, Anda perlu melakukan loadstring untuk memanggil library utama. Berikut kodenya:

lua
local librarytriplesixxx = loadstring(game:HttpGet("https://triplesixxx-xyz.vercel.app/api/647ca5e7/loader.lua"))()

Selanjutnya, Anda dapat langsung membuat variabel Window untuk menyimpan GUI Anda.

2. Konfigurasi Window (Window Settings)

Konfigurasi `Window` menentukan tampilan awal GUI, termasuk efek salju (rainsnow), warna, folder konfigurasi, dsb.

lua
local Window = librarytriplesixxx:Window({ Icon = "rbxassetid://107816383763884", -- Ikon header kiri Name = addcolor("Triple", 255, 0, 0) .. " sixx | Community", Subtitle = addcolor("Example Gui", 255, 165, 0), Version = "V 2.0 | NEW UPDATED!", MarqueeTitle = "Subtitle", -- "Name" atau "Subtitle" akan berjalan mengetik Color = Color3.fromRGB(255, 0, 0), -- Warna aksen utama Background = "rainsnow", -- Efek salju di panel belakang makeconfig = "Triplesixxx/Saved", -- Lokasi custom pembuat folder config lokal SnowColor = Color3.fromRGB(255, 255, 255), SnowSpeed = 1, SnowSize = 3, SnowDirection = "Vertical", -- Bisa "Vertical", "Horizontal Left", "Horizontal Right" Keybind = Enum.KeyCode.RightControl })

3. Tabs & Sections

Untuk mengelompokkan elemen UI Anda, buatlah `Tab`, lalu tambahkan `Section` di dalamnya.

lua
local MainTab = Window:AddTab({ Name = "Utama", Icon = "home" -- Bisa menggunakan icon lucide atau ID rbxasset }) local MainSection = MainTab:AddSection("Fitur Pertarungan")

4. Buttons, Toggles, & Panels

Tombol untuk melakukan aksi ekseskusi, dan Toggle untuk aksi yang dapat diaktifkan/matikan (menyimpan opsi config otomatis).

lua
-- Tombol Biasa MainSection:AddButton({ Title = "Klik Saya!", Callback = function() Notify("Tombol Ditekan!") end }) -- Toggle Switch MainSection:AddToggle({ Title = "Auto Farm", Default = false, -- otomatis load data sebelumnya jika ada di folder Content = "Menyala otomatis hajar monster.", Callback = function(v) print("Toggle is now ", v) end }) -- Panel dengan Kolom Teks dan Eksekusi Callback MainSection:AddPanel({ Title = "Code Execution", Placeholder = "Teks Disini...", Button = "Mulai", SubButton = "Hentikan", Callback = function(text) print("Start: " .. text); end, SubCallback = function() print("Stop button pressed"); end })

5. Sliders & Text Inputs

Slider biasa digunakan untuk mengatur nilai, dan Text Input untuk mengisi data string atau angka tunggal yang akan di simpan.

lua
-- Slider MainSection:AddSlider({ Title = "WalkSpeed", Min = 16, Max = 300, Default = 50, Increment = 1, Content = "Kecepatan berjalan", Callback = function(val) game.Players.LocalPlayer.Character.Humanoid.WalkSpeed = val end }) -- Input Box MainSection:AddInput({ Title = "JumpPower (Hack)", Placeholder = "50", Default = "50", Callback = function(textValue) print("Jump set to: " .. textValue) end })

6. Dropdowns

Cocok untuk seleksi tipe lokasi teleportasi, mode serangan, dll. Mendukung Multi-seleksi (memilih lebih dari satu item).

lua
MainSection:AddDropdown({ Title = "Teleport Area", Options = {"Spawn", "Shop", "Arena"}, Default = "Spawn", Content = "Pilih tempat favorit", Callback = function(val) Notify("Teleporting to: " .. val) end })

7. Customisasi Visual (Images, Colors, addcolor)

Berbagai komponen pendukung seperti Pembatas Garis, Image Label, Custom Color Picker, dan Fungsi pembuat font berwarna `addcolor`.

lua
-- Pembatas Garis Berclear Gradient MainSection:AddDivider() -- Memunculkan Image khusus berukuran besar di Frame MainSection:addrbxasset("rbxassetid://14204231522, name = Foto Profil") -- Color Picker Biasa (Ada Palet Default) MainSection:AddColorPicker({ Title = "Highlight ESP Color", Default = Color3.fromRGB(255, 0, 0), Callback = function(color3) print("Warna baru dipilih!") end }) -- FUNGSI GLOBAL addcolor(text, R, G, B) -- Sangat Berguna! local TeksKeren = addcolor("MERAH!!", 255, 0, 0) Notify(TeksKeren) -- akan mengeluarkan teks MERAH!! berwarna merah sungguhan di UI

8. Sistem Notifikasi (Notify)

Fitur notifikasi terbaru yang mensupport Ikon bebas, RichText berwarna, dan transisi UI yang tidak akan menutupi atau crash.

lua
-- Pemanggilan Basic Notify("Operasi berhasil!", 3 --[[delay timeout]], Color3.fromRGB(0,255,0), "Sukses", "System") -- Pemanggilan Lanjutan menggunakan Tabel (Direkomendasikan) librarytriplesixxx:MakeNotify({ Title = addcolor("Peringatan!", 255, 0, 0), Description = " Server Warning", Content = "Banned terdeteksi, bersiap keluar...", Icon = "rbxassetid://107816383763884", -- atau icon string "home", "eyes" Time = 4, -- Durasi Muncul Delay = 6 -- Durasi Bertahan })
